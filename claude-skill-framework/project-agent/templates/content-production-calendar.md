## Editorial Workflow

### Content Creation Process

```mermaid
flowchart TD
    A[Idea Generation] --> B[Content Brief]
    B --> C[Research & Outline]
    C --> D[First Draft]
    D --> E[Self Review]
    E --> F[Peer Review]
    F --> G{Approved?}
    G -->|No| H[Revisions]
    H --> D
    G -->|Yes| I[Supervisor Review]
    I --> J{Final Approval?}
    J -->|No| K[Final Edits]
    K --> D
    J -->|Yes| L[Schedule/Publish]
    L --> M[Promote]
    M --> N[Monitor Performance]
```

### Review Checklist

- [ ] Content aligns with brand voice
- [ ] SEO keywords included
- [ ] Images/media optimized
- [ ] Links working
- [ ] Call-to-action included
- [ ] Spelling/grammar checked
- [ ] Platform requirements met
