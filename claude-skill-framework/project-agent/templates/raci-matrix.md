## Matrix Overview

The RACI matrix defines responsibility assignments across all project tasks and deliverables to ensure clear ownership and accountability.

| [Task 12] | I | C | C | I | |

### 1.2 RACI Legend
| Symbol | Meaning | Description |
|------|---------|-------------|
| **R** | Responsible | Does the work or leads the task |
| **A** | Accountable | Final decision authority, only oversight |
| **C** | Consulted | Provides input/advice, Two-way communication |
| **I** | Informed | Kept in the loop, Receives updates on progress |
