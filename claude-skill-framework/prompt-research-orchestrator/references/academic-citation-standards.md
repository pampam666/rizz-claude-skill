# Academic Citation Standards

> Citation formatting guide for prompt engineering research

---

## Citation Style: APA 7th Edition

All citations in Prompt Research Orchestrator reports follow APA 7th Edition format.

---

## In-Text Citations

### Single Author
```
(Wei, 2022) or Wei (2022) argues that...
```

### Two Authors
```
(Wei & Wang, 2022) or Wei and Wang (2022) found...
```

### Three or More Authors
```
(Wei et al., 2022) or Wei et al. (2022) demonstrated...
```

### Multiple Sources
```
(Wei, 2022; Yao et al., 2023)
```

### Direct Quotes
```
(Wei, 2022, p. 3) for page-specific quotes
```

---

## Reference List Formats

### 1. Journal Articles

**Format:**
```
Author, A. A., & Author, B. B. (Year). Title of article. Journal Name, Volume(Issue), Page-Page. DOI
```

**Example:**
```
Wei, J., Wang, X., Schuurmans, D., Bosma, M., Ichter, B., Xia, F., Chi, E., Le, Q., & Zhou, D. (2022). Chain-of-Thought Prompting Elicits Reasoning in Large Language Models. Advances in Neural Information Processing Systems, 35, 24824-24837.
```

---

### 2. arXiv Preprints

**Format:**
```
Author, A. A., Author, B. B., & Author, C. C. (Year). Title of paper. arXiv preprint arXiv:XXXX.XXXXX.
URL: https://arxiv.org/abs/XXXX.XXXXX
```

**Example:**
```
Yao, S., Yu, D., Zhao, J., Shafran, I., Griffiths, T. L., Cao, Y., & Narasimhan, K. (2023). Tree of Thoughts: Deliberate Problem Solving with Large Language Models. arXiv preprint arXiv:2305.10601.
URL: https://arxiv.org/abs/2305.10601
```

---

### 3. Conference Papers

**Format:**
```
Author, A. A. (Year, Month). Title of paper. In Proceedings of Conference Name (pp. Page-Page). Publisher.
```

**Example:**
```
Yao, S., Zhao, J., Yu, D., Du, D., Shafran, I., Narasimhan, K., & Cao, Y. (2022, December). ReAct: Synergizing Reasoning and Acting in Language Models. In International Conference on Learning Representations.
```

---

### 4. Online Documentation

**Format:**
```
Organization. (Year, Month Day). Title of page. Site Name.
URL: [full URL] Accessed: [date]
```

**Example:**
```
Anthropic. (2024, January 15). Prompt Engineering Guide. Anthropic Documentation.
URL: https://docs.anthropic.com/claude/docs/prompt-engineering Accessed: 2024-01-20
```

---

### 5. Technical Blogs

**Format:**
```
Author, A. A. (Year, Month Day). Title of post [Blog post]. Blog Name.
URL: [full URL] Accessed: [date]
```

**Example:**
```
Brown, T. (2022, March 15). Understanding Chain-of-Thought Prompting [Blog post]. OpenAI Blog.
URL: https://openai.com/blog/chain-of-thought Accessed: 2024-01-20
```

---

### 6. GitHub Repositories

**Format:**
```
Author, A. A. (Year). Title of repository [Computer software]. GitHub.
URL: [full URL] Accessed: [date]
```

**Example:**
```
PromptingGuide.ai. (2024). Prompt Engineering Guide [Computer software]. GitHub.
URL: https://github.com/dair-ai/Prompt-Engineering-Guide Accessed: 2024-01-20
```

---

## Special Cases

### AI-Generated Content

When citing AI-generated content:
```
Claude. (2024). Response to prompt about chain-of-thought prompting [Large language model]. Anthropic.
Accessed: 2024-01-20
```

### Personal Communication
```
(Smith, personal communication, January 15, 2024)
```
Note: Not included in reference list.

### Secondary Sources
```
(Cited in Wei, 2022)
```

## Reference List Organization

### Alphabetical Order
Sort by first author's last name.

### Hanging Indent
```
Wei, J., Wang, X., Schuurmans, D., Bosma, M., Ichter, B., Xia, F., Chi, E., 
    Le, Q., & Zhou, D. (2022). Chain-of-Thought Prompting Elicits 
    Reasoning in Large Language Models. Advances in Neural Information 
    Processing Systems, 35, 24824-24837.
```

---

## Quality Checklist

- [ ] All in-text citations have corresponding reference list entries
- [ ] All references formatted in APA style
- [ ] URLs included for online sources
- [ ] Access dates included for online sources
- [ ] Authors listed correctly (use et al. for 4+)
- [ ] Year of publication included
- [ ] Alphabetical order maintained
- [ ] Hanging indents applied

---

## Common Errors to Avoid

| Error | Incorrect | Correct |
|-------|----------|---------|
| Missing access date | URL: [link] | URL: [link] Accessed: [date] |
| Wrong et al. usage | Wei et al. (2022) for 2 authors | Wei and Wang (2022) |
| Missing DOI | Journal article without DOI | Include DOI when available |
| Inconsistent style | Mixing citation styles | Use APA consistently |
