# Report Generator

Specialist skill for generating professionally formatted DOCX research reports with academic citations.

## Mission

Generate comprehensive DOCX reports from research and analysis artifacts, applying proper formatting, citations, and templates.

## Activation Protocol

### Automatic Triggers
- Analysis phase completion detected
- Report generation requested
- DOCX output required

### Input Requirements

| Input | Source | Required |
|-------|--------|----------|
| `research_artifacts` | workflow_state.json | Yes |
| `analysis_artifacts` | workflow_state.json | Yes |
| `selected_toc` | workflow_state.json | Yes |

### Phase 3: Citation Formatting

Use APA style for all citations:

**Format:**
```
Author, A. A. (Year). Title of work. Source.
URL: [link] Accessed: [date]
```

**Example:**
```
Wei, J., Wang, X., Schuurmans, D., Bosma, M., Ichter, B., Xia, F., Chi, E., Le, Q., & Zhou, D. (2022). Chain-of-Thought Prompting Elicits Reasoning in Large Language Models. arXiv preprint arXiv:2201.11903.
URL: https://arxiv.org/abs/2201.11903 Accessed: 2024-01-15
```

## DOCX Generation Script

Execute `scripts/generate_docx.py`:

```python
# Pseudo-code for DOCX generation
def generate_report(workflow_state, output_path):
    """
    Generate DOCX report from workflow state.
    
    Args:
        workflow_state: Path to workflow_state.json
        output_path: Path for output .docx file
    
    Returns:
        Path to generated DOCX file
    """
    # Load data
    data = load_workflow_state(workflow_state)
    
    # Create document
    doc = Document()
    
    # Add title page
    add_title_page(doc, data)
    
    # Add sections based on TOC
    for section in data['selected_toc']['sections']:
        add_section(doc, section, data)
    
    # Add references
    add_references(doc, data['research_artifacts']['sources'])
    
    # Save document
    doc.save(output_path)
    
    return output_path
```

## Citation Execution

Execute `scripts/format_citations.py`:

```python
def format_citations(sources):
    """
    Format sources into APA citations.
    
    Args:
        sources: List of source dictionaries
    
    Returns:
        List of formatted citation strings
    """
    citations = []
    for source in sources:
        citation = format_apa(source)
        citations.append({
            'apa_format': citation,
            'in_text': f"({source['authors']}, {source['year']})",
            'source_url': source['url']
        })
    return citations
```
