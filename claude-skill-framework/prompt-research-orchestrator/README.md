# Prompt Research Orchestrator

> **Multi-Skill Framework for Scientist-Level Prompt Engineering Research**

---

## Quick Start

```
User: "Compare Chain-of-Thought vs ReAct prompting frameworks with pros/cons table, ease of use, and LLM thinking process analysis."
```

The framework will:
1. Generate 3-5 TOC options for your approval
2. Research frameworks from PromptingGuide.ai + Anthropic docs
3. Build comprehensive comparison tables
4. Generate formatted DOCX with academic citations

---

## Framework Components

| Skill | Purpose |
|-------|---------|
| **mother-orchestrator** | Single user interface, TOC proposals, workflow coordination |
| **framework-researcher** | Web research, framework analysis, source gathering |
| **comparison-analyst** | Comparison tables, metrics, benchmarking |
| **report-generator** | DOCX generation, citations, formatting |

---

## Workflow

```
Request → TOC Proposal → Research → Analysis → Report → Delivery
```

## Metrics Included

| Metric | Description |
|--------|-------------|
| Pros/Cons | Structured advantages and disadvantages |
| Ease of Use | Learning curve and implementation complexity |
| LLM Thinking Process | How each framework affects model reasoning |
| Scalability | Enterprise and production readiness |
| Grok/Claude Compatibility | Cross-platform support |
| Benchmarks | Performance comparisons from literature |
| Citation Quality | Academic source rigor |
