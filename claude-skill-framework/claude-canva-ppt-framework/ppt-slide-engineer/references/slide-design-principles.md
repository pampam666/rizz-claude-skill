# Slide Design Principles

This document outlines the fundamental principles for creating professional, effective presentation slides as part of the ppt-slide-engineer skill.

## Core Design Philosophy

### Simplicity Over Complexity
- Less text on often = more impactful slides
- White space improves readability
- One main idea per slide
- Maximum 6 bullet points per slide
- Use progressive disclosure for detailed information

### Visual Hierarchy
- Establish clear focal points using size, color, and position
- Use contrast to differentiate elements
- Align content to natural reading patterns
- Create visual flow from top-left to bottom-right

### Consistency
- Maintain uniform styling throughout
- Use consistent color palette
- Apply same font families across all slides
- Keep spacing and alignment consistent

## Slide Type Guidelines

### Title Slide
- Large, bold typography for main title
- Subtitle in smaller text below
- Background should set tone (gradient, image, solid color)
- Optional decorative elements (shapes, patterns)
- Include date/presenter name if applicable

### Agenda Slide
- Clear heading: "Agenda" or "Overview" or "Today's Topics"
- List 3-5 main topics
- Use icons or numbers for visual interest
- Keep descriptions minimal
- Consider timeline format for workshops

### Content Slides
- Clear, descriptive title
- 3-5 bullet points maximum
- Supporting visuals (icons, images)
- Consistent text hierarchy
- Speaker notes for presenter guidance
- Avoid wall-of-text approach

### Data Slides
- Prominent chart or graph as focal point
- Clear title and labels
- Legend when necessary
- Key insight callout
- Source citation for credibility
- Use appropriate chart type for data

### Conclusion Slide
- Summary heading
- 3-5 key takeaways
- Clear call-to-action
- Contact/next steps information
- Thank you slide
- Professional closing

## Typography Guidelines

### Font Pairing
- **Heading Font**: Montserrat, Poppins, Arial Black (sans-serif)
- **Body Font**: Open Sans, Lato, Roboto (sans-serif)
- Maximum 2 font families per presentation

### Size Hierarchy
| Element | Size | Weight | Usage |
|--------|------|--------|-------|
| Title Slide Title | 60-72pt | Bold | Main presentation title |
| Slide Title | 36-44pt | Bold | Content slide headers |
| Subtitle | 24-28pt | Regular | Supporting text |
| Body Text | 18-24pt | Regular | Paragraphs, descriptions |
| Bullet Points | 16-20pt | Regular | List items |
| Caption | 12-14pt | Light | Image credits, notes |

### Line Spacing
- **Title**: 1.2x line height
- **Body**: 1.5x line height
- **Bullets**: 1.0 line height
- **Caption**: 1.0 line height

## Color Application

### Color Distribution
- **Primary**: 60% - Main elements, titles, key accents
- **Secondary**: 30% - Supporting elements, backgrounds
- **Accent**: 10% - Call-to-action, highlights
- **Neutral**: 60% - Text, backgrounds, dividers

### Background Colors
- **Light backgrounds**: White (#FFFFFF), Light Gray (#F5F5F5)
- **Dark backgrounds**: Dark colors from brand palette
- **Gradient backgrounds**: Primary to secondary color transition

### Text Colors
- **Dark text**: On light backgrounds
- **Light text**: On dark backgrounds
- **Accent text**: For emphasis and key points

## Visual Elements

### Icons
- Use consistent icon style throughout
- Recommended icon libraries: Font Awesome, Material Icons
- Size: 24-32px for inline icons
- Color: Match brand accent color
- Use icons to enhance, not decorate

### Images
- High-quality, relevant images
- Avoid stock photos that look generic
- Consider custom illustrations
- Optimize for presentation context
- Maintain aspect ratio consistency

### Charts
- **Bar Charts**: Comparisons, rankings
- **Pie Charts**: Proportions, percentages
- **Line Charts**: Trends over time
- **Area Charts**: Cumulative values
- Keep charts simple with clear labels
- Use brand colors in charts

## Layout Patterns

### Single Column
- Title at top
- Bullet points below
- Full width utilization
- Good for text-heavy content

### Two Column
- Split screen vertically or horizontally
- Compare/contrast content
- Image + text combinations
- Good for comparisons

### Grid Layout
- Multiple equal elements
- Feature lists
- Team introductions
- Product features
- Maintain alignment

## Animation Guidelines
### Entrance Animations
- **Fade In**: Subtle, professional
- **Slide In**: Directional flow
- **Appear**: Instant, no animation
- Avoid: Spin, bounce, flashy effects

### Transition Animations
- **Fade**: Smooth between slides
- **Slide**: Directional movement
- **None**: Instant transition
- Keep consistent throughout presentation

## Accessibility Considerations
- Minimum font size: 18pt for readability
- High contrast between text and background
- Avoid red/green combinations (color blindness)
- Clear, simple language
- Alt text for images
- Logical reading order

## Quality Checklist
- [ ] Consistent font usage
- [ ] Color palette applied throughout
- [ ] All images high quality
- [ ] Charts are clearly labeled
- [ ] Text is readable
- [ ] Alignment is consistent
- [ ] Animations are subtle
- [ ] Speaker notes complete
- [ ] Spelling and grammar checked
- [ ] Brand guidelines followed
